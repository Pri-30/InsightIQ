# InsightIQ 🚀

Full-stack AI-powered analytics platform that processes documents (PDF, Excel, text) and generates intelligent summaries, visual insights, and data-driven charts using AI with a fallback mechanism for reliable outputs.

## 🔥 Features
- PDF, Excel, and text file processing
- AI-generated summaries and insights
- Data visualization and charts
- Fallback system for reliable responses when AI services fail
- Full-stack architecture (Frontend + Backend)

## 🧠 Tech Stack
- Frontend: React (Vite)
- Backend: Python (Flask/FastAPI)
- AI: LLM-based processing
- Data Processing: Pandas, custom parsers
